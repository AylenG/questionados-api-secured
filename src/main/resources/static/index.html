Questionados

Una api de preguntas y respuestas tipo 'Preguntados' donde se podrá:

  • Crear categorías de preguntas
  • Obtener todas las categorías
  • Crear preguntas
  • Obtener todas las preguntas
  • Obtener pregunta aleatoria
  • Validar la respuesta de alguna pregunta
  • Repositorio
  • Swagger
